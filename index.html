O que significa HTML?

(voltar)

O HTML foi criado pelo britânico br Tim Berners-Lee. O significado das letras HTML vem do inicio de cada palavra em inglês que seria: HiperText Markup Language, traduzindo para o português fica: Linguagem de Marcação de Hipertexto. Entende-se que o HTML é o componente básico da web, ele permite inserir o conteúdo e estabelecer a estrutura básica de um website. Portanto, ele serve para dar significado e organizar as informações de uma página na web. Sem isso, o navegador não saberia exibir textos como elementos ou carregar imagens e outros conteúdos.

Pata melhor compreensão, os hipertextos são conjuntos de elementos conectados. Esses podem ser palavras, imagens, vídeos, documento, etc. Quando conectados, formam uma rede de informações que permite a comunicação de dados, organizando conhecimentos e guardando informações.

Repare que ao visitar uma página simples na web, você pode perceber que existem diferentes distribuições e tamanhos para títulos, parágrafos, imagens, vídeos e qualquer outro elemento. Essa estrutura é estabelecida através do HTML. A título de curiosidade, no inicio da web, era comum encontrar sites apenas contendo textos e imagens simples, com estrutura básica e sem estilizações. Porém, nos dias atuais, muito dificilmente você encontrará sites que possuam apenas elementos HTML. Portanto, podemos considerar o HTML o “esqueleto” da sua página.

Imagine então que além do esqueleto, é necessário ter o corpo. Para isso, temos então as linguagens CSS e o JavaScript, que em conjunto com HTML, formam a base para todos os websites atuais.

Em breve falarei mais a respeito dessas outras linguagens!

As tags que eu aprendi nas aulas da DIO

(voltar)

Para falar a respeito das tags é preciso entender que todas as páginas HTML possuem uma série de elementos, que consistem num conjunto de tags e atributos. Os elementos HTML são os tijolos de construção de uma página da internet. Uma tag diz para o navegador onde um elemento começa e termina, enquanto um atributo descreve as características de um elemento.

As três principais partes de um elemento são:

  • Tag de abertura – usada para dizer onde um elemento começa a ter efeito. A tag é cercada de colchetes angulares para abertura e fechamento. Por exemplo, use a tag de abertura <p> para criar um parágrafo.
  • Conteúdo – essa é a parte que os usuários verão.
  • Tag de fechamento – igual à tag de abertura, mas com uma barra antes do nome do elemento. Por exemplo, </p> para encerrar um parágrafo.

A seguir listo as tags importantes aprendidas durante as aulas e um complemento a mais depois de ter feito uma pesquisa a respeito:

  1. As tags: <h1>, <h2>, <h3>, <h4>, <h5> e <h6> – são tags para definir um título e subtítulos, variando de 1 a 6, sendo h1 o título mais importante e h6 o de menor importância. Além disso o texto assume automaticamente a propriedade negrito. Veja abaixo o tamanho da fonte de cada nível:
    • O marcador <h1></h1> definine um título com fonte tamanho 24 pontos;
    • O marcador <h2></h2>definine um título com fonte tamanho 18 pontos;
    • O marcador <h3></h3> definine um título com fonte tamanho 14 ponto;
    • O marcador <h4></h4> definine um título com fonte tamanho 12 pontos;
    • O marcador <h5></h5> definine um título com fonte tamanho 10 pontos;
    • O marcador <h6></h6> definine um título com fonte tamanho 8 pontos.

  2. A tag <p> - é uma tag usada para definir um parágrafo. Com essa tag pode estabelecido alguns tipos de alinhamentos horizontais, são eles:
    • À esquerda: <p> Parágrafo alinhado à esquerda.
    • À direita: <p align="right"> Parágrafo alinhado à direita.
    • No centro: <p align="center"> Parágrafo alinhado no centro.
    • Justificado: <p align="center"> Parágrafo alinhado no centro.

  3. A tag <mark> - é usado para destacar partes do texto, como se fosse um marca texto de cor amarelo.

  4. A tag <small> - é uma tag usada para comentários laterais e letras pequenas, como direitos autorais e texto legal, independente de sua apresentação estilizada. Por padrão, ele renderiza o texto dentro dele um tamanho de fonte menor.

  5. A tag <i> - coloca o texto em itálico.

  6. A tag <u> - é utilizado para sublinhar o texto.

  7. A tag <strong> - do inglês "Strong Element" ou fonte forte é utilizado para destacar em negrito um intervalo definido de um texto com maior importância.

  8. A tag <ol> - é uma tag que cria uma lista ordenada. Ela coloca os números em ordem crescente, ou seja, nesta lista os tópicos começam com os números 1,2,3, etc. Pode usar essa tag, de acordo com a sequência de numeração da sua escolha:
    • <ol type="a"> - sendo a para letras minúsculas;
    • <ol type="A"> - sendo A para letras maiúsculas;
    • <ol type="i"> - sendo i para algarismos romanos minúsculas;
    • <ol type="I"> - sendo I para algarismos romanos maiúsculas;
    • <ol type="1"> - sendo 1 para números (padrão).

  9. A tag <ul> - é uma tag que cria uma lista não ordenada, ou seja, ela é representada por um caractere chamado marcador bullet •.

  10. A tag <li> - é um item de lista que é usado dentro da lista ordenada e lista não ordenada. Em menus e listas não ordenadas, os itens de lista geralmente são exibidos usando marcadores. Em listas ordenadas, eles geralmente são exibidos com um contador crescente à esquerda, como um número ou uma letra.

  11. A tag <a> - é uma tag chamada de âncora e funciona com o atributo href. Ele cria um hiperlink para as páginas web, arquivos, endereços de emails, locais na mesma página ou endereços na URL. O conteúdo dentro de cada âncora precisará indicar o destino do link. Para isso, use o atributo href e coloque dentro das aspas a destino desejado. Para saber onde tem a âncora, basta visulizar o texto na página de cor azul e sublinado, e ao clicar irá fazer o carregamento do link no navegador. O link poderá ser exibido como uma guia, janela ou frame. A seguir veja os tipos de configurações da abertura desse link.

    12. Dentro da âncora use o atributo target="_nome" com uma das opções abaixo, conforme necessidade:

    • _self: o contexto de navegação atual, o link direciona para dentro do próprio site.
    • _blank: geralmente uma nova guia, mas os usuários podem configurar os navegadores para abrir uma nova janela.
    • _parent: especifica onde a página vinculada ou a resposta do formulário será aberta. O site abre a navegação no pai (quadro, guia ou janela). Se nãto tiver nenhum pai, ele se comporta como um _self.
    • _top: atributo especifica que a página vinculada ou a resposta do formulário será aberta no quadro superior. O quadro superior é a guia e a janela completas do navegador.

  12. A tag <hr> - vem do inglês "Horizontal Rule". Ela é usada para inserir uma linha horizontal numa página HTML.

  13. A tag <sub> - significa subscrito é utilizado para escrever um texto com letras menores e com o alinhamento abaixo das que do resto do conteúdo. Por exemplo: C6H6

  14. A tag <sup> - significa sobrescrito é utilizado para escrever um texto com letras menores e com o alinhamento acima das que do resto do conteúdo. Por exemplo: a2=b2+c2

  15. A tag <blockquote> - cria uma citação. Essa tag destaca o texto com espaçamento de margem.

Novas tags sugeridas pela DIO

(voltar)

As tags mencionadas abaixo, são uma sugestão da DIO para estudo e aperfeiçoamento individual do aluno do bootcamp com o HTML. Veja a seguir quais são as tags:

  1. <font> - a tag font costumava ser usada no HTML 4 para especificar o tamanho da fonte, a cor e a face do seu conteúdo.
    • Atenção: este recurso não é mais recomendado. Embora alguns navegadores ainda possam suportá-lo, ele pode já ter sido removido dos padrões da Web relevantes, pode estar em processo de eliminação ou pode ser mantido apenas para fins de compatibilidade. Esteja ciente de que esse recurso pode deixar de funcionar a qualquer momento.
    • Aviso: esteja ciente de que esse recurso pode deixar de funcionar a qualquer momento.
  2. <del> - vem do inglês "Deleted Text", entenda essa tag como um texto excluído ou censurado. Ela é utilizada para indicar que um texto foi suprimido, apesar de, aparecer riscado, para que o leitor tenha ciência da ocorrência. A critério do editor, após o texto contido dentro da tag <del></del>, pode-se colocar uma informação recente usando-se a tag <ins></ins>, em substituição ao texto suprimido, que aparecerá destacado.
  3. <b> - essa tag b de "bold" significa negrito. É utilizado para destacar alguma parte de um texto de forma mais forte do que o resto daquele texto.
  4. <abbr> - representa uma abreviação ou acrônimo. Isso informa ao usuário o que a abreviação ou sigla significa. É útil para fornecer informações aos navegadores, leitores de telas, buscadores, sites de tradução como também verificadores de ortografia.
    • O atributo opcional title pode fornecer uma expansão para a abreviação ou acrônimo quando uma expansão completa não estiver presente. Isso fornece uma dica para os agentes de usuário sobre como anunciar/exibir o conteúdo enquanto informa a todos os usuários o que a abreviação significa. Se presente, title deve conter esta descrição completa e nada mais.
    • O title atributo tem um significado semântico específico quando usado com o <abbr> elemento; deve conter uma descrição legível completa ou expansão da abreviatura. Este texto é frequentemente apresentado pelos navegadores como uma dica de ferramenta quando o cursor do mouse passa sobre o elemento.
    • Cada <abbr> elemento que você usa é independente de todos os outros; fornecer um title não anexa automaticamente o mesmo texto de expansão a outros com o mesmo texto de conteúdo.
